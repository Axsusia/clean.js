##random
랜덤 문자열을 반환하는 함수입니다.
```
//코드
var randomString = clean.string.random(15);
console.log('랜덤 문자열 =>' + randomString);

//결과
랜덤 문자열 => tGASq7LwUj6I591
```

##randomColor

랜덤 색상 문자열을 반환하는 함수입니다.

```
//코드
var randomColorString = clean.string.randomColor();
console.log('랜덤 색상 문자열 => ' + randomColorString);

//결과
랜덤 색상 문자열 => #3ad3e7
```
